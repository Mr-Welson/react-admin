

## 问题记录

1. Module build failed (from ./node_modules/less-loader/dist/cjs.js):TypeError: this.getOptions is not a function
   原因：8.0+ 版本 less-loader 的配置方式发生了变化，导致报错。最简单的解决办法：安装低版本 ` yarn add less-loader@5.0.0 -D `
     其他方案：https://segmentfault.com/a/1190000039190699
